5. S-CURVE ADOPTION — MUST USE NORMALIZED LOGISTIC k=5 (NEVER PRE-COMPUTE IN PYTHON):

   The forecast server uses this exact algorithm (ramp_scurve):
     tr = year_index / time_to_peak          # normalized 0..1
     sig_0 = 1/(1+exp(2.5))                  # ≈ 0.07585 — constant
     sig_1 = 1/(1+exp(-2.5))                 # ≈ 0.92415 — constant
     sig   = 1/(1+exp(-5*(tr - 0.5)))
     share = start + (peak - start) * (sig - sig_0) / (sig_1 - sig_0)
   This guarantees share = start exactly at year 0, share = peak exactly at final year.

   The Excel formula that matches this EXACTLY (COLUMN()-2 = year_index for data starting at col B):
     =start_ref+(peak_ref-start_ref)*(1/(1+EXP(-5*((COLUMN()-2)/(end_yr_ref-launch_yr_ref)-0.5)))-1/(1+EXP(2.5)))/(1/(1+EXP(-2.5))-1/(1+EXP(2.5)))

   Where:
     • start_ref      = absolute Inputs cell for starting share  (e.g. Inputs!$B$18)
     • peak_ref       = absolute Inputs cell for peak share      (e.g. Inputs!$B$17)
     • end_yr_ref     = absolute Inputs cell for peak/end year   (e.g. Inputs!$B$8)
     • launch_yr_ref  = absolute Inputs cell for launch year     (e.g. Inputs!$B$7)
     • 1/(1+EXP(2.5)) and 1/(1+EXP(-2.5)) are constants — hardcode them as-is.

   Use the SAME formula string for every year column — Excel's COLUMN() evaluates per cell.
   NEVER compute S-curve values in Python and write them as literal floats.

   Example Python code to apply:
     _n   = "(Inputs!$B$8-Inputs!$B$7)"           # end_year - launch_year
     _x   = f"(COLUMN()-2)/{_n}"                  # normalized index 0..1
     _sig = f"1/(1+EXP(-5*({_x}-0.5)))"
     _s0  = "1/(1+EXP(2.5))"
     _s1  = "1/(1+EXP(-2.5))"
     scurve_class   = f"=Inputs!$B$start_cs+( Inputs!$B$peak_cs-Inputs!$B$start_cs)*({_sig}-{_s0})/({_s1}-{_s0})"
     scurve_product = f"=Inputs!$B$start_ps+(Inputs!$B$peak_ps-Inputs!$B$start_ps)*({_sig}-{_s0})/({_s1}-{_s0})"
     for i in range(n_years):
         ws.cell(row=class_row,   column=2+i).value = scurve_class
         ws.cell(row=product_row, column=2+i).value = scurve_product
