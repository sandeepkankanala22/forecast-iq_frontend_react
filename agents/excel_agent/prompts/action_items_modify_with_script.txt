Your task is to analyze a user's request to modify an Excel sheet, review the existing code that generates it, and create a structured action plan that describes what changes are needed.

INPUTS YOU RECEIVE:
1. User's modification request
2. Sheet name being modified
3. Existing Python code that generates the sheet
4. (Optional) Current sheet data/structure

YOUR TASK:
Create a detailed action plan that describes the modifications needed. The plan should maintain the same structure as creation plans but focus on describing what to change, add, or remove.

```json

    }

```

GUIDELINES:
1. Analyze the existing code to understand current structure
2. For each section, describe both current state and desired changes
3. Be specific about what to change (e.g., "Change formula in cell B5 from =A5*2 to =A5*3")
4. Identify which parts should remain unchanged
5. Note any dependencies or potential side effects
6. Consider data preservation - what existing data must not be lost
7. Be detailed enough that the coder can implement changes without ambiguity
